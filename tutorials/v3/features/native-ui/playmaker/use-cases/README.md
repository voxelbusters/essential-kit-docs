# NativeUI Use Cases

Quick-start guides for native dialogs and pickers using PlayMaker custom actions.

## Available Use Cases

### 1. [Confirmation Dialog](use-case-1-confirmation-dialog.md)

- **What it does:** Show alert and branch on button
- **Actions:** 1 (ShowAlert)

### 2. [Text Input Dialog](use-case-2-text-input.md)

- **What it does:** Prompt for text input and validate
- **Actions:** 1 (ShowAlert with TextInput)

### 3. [Date Selection](use-case-3-date-selection.md)

- **What it does:** Pick date/time and handle select/cancel
- **Actions:** 1 (ShowDatePicker)

## Quick Action Reference
| Action | Purpose |
|--------|---------|
| NativeUIShowAlert | Display native alerts with buttons and optional single text input |
| NativeUIShowDatePicker | Display native date/time picker |
