# Media Services Tutorial - Introduction

Welcome to the Media Services tutorial! This tutorial is part of the Essential Kit tutorial series from Voxel Busters.

## What You'll Learn

In this comprehensive tutorial series, you'll master the Essential Kit's Media Services feature for Unity cross-platform development:

- How to check gallery and camera access permissions
- Selecting images, videos, and other media from the device gallery  
- Capturing photos and videos using the device camera
- Saving media content to the device gallery
- Processing media content in various formats (Texture2D, raw data, file paths)
- Advanced configuration and error handling techniques

## Why Media Services Matter for Unity Mobile Game Developers

Media Services are essential for Unity mobile game development, enabling seamless integration with device cameras and photo galleries. This feature allows players to personalize their gaming experience by importing custom content, sharing achievements, or creating user-generated content directly within your Unity iOS and Unity Android applications.

Common use cases include:
- Profile picture customization in social games
- Screenshot sharing functionality
- Custom avatar creation from photos
- User-generated content campaigns
- In-game photo contests and galleries

## Prerequisites

- Unity 2021.3.45f1 or compatible version
- Essential Kit properly installed and configured
- Basic understanding of Unity C# scripting
- Mobile device for testing (iOS/Android)

## Platform Setup

Essential Kit handles most native setup automatically for both Unity iOS and Unity Android platforms, including:

- Framework dependencies and permissions management
- Native library integration and manifest entries  
- Cross-platform API abstraction layer
- Automatic permission dialog handling

This automation eliminates the complexity of manual native development setup! You only need to ensure:

- Your app's privacy policy covers media access (required by app stores)
- Store compliance requirements are met for your target platforms
