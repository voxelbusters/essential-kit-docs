# Native UI - Introduction

Welcome to the Native UI tutorial series! This tutorial is part of the Essential Kit tutorial series from Voxel Busters.

Native UI components provide essential user interface elements that seamlessly integrate with the native look and feel of iOS and Android platforms. These components are crucial for creating polished Unity mobile games that feel natural to users on each platform.

## What You'll Learn

- How to display native alert dialogs with custom buttons and text input fields
- How to implement native date pickers for scheduling and time-based features
- How to handle user interactions and callbacks from native UI components
- How to configure and customize native UI elements for your Unity mobile game
- Best practices for implementing native UI in cross-platform Unity projects

## Why Native UI Matters for Unity Mobile Game Developers

Native UI components are essential for Unity mobile game development because they provide platform-specific interface elements that users expect. Players are familiar with the standard alert dialogs and date pickers on their devices, so using native components creates a more polished and professional experience compared to custom Unity UI alternatives.

Common use cases include:
- Confirmation dialogs for important actions (purchases, deletions, etc.)
- User input forms for player names, passwords, or feedback
- Date/time selection for scheduling game events or tournaments
- System notifications and alerts that match platform conventions

## Prerequisites

- Unity 2021.3.45f1 or compatible version
- Essential Kit properly configured in your Unity project
- Basic understanding of Unity scripting with C#
- Target platform set to iOS or Android

## Platform Setup

**Essential Kit handles most native setup automatically!** This includes:
- Native framework integration for iOS and Android
- Required permissions and manifest entries
- Platform-specific UI styling and behavior
- Cross-platform API abstraction

You only need to handle:
- Privacy policy compliance for user data collection
- App store requirements and guidelines
- Platform-specific design considerations

This automation is one of Essential Kit's major value propositions - you get native functionality without the complex setup process!
