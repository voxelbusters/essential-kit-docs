# Sharing Services Tutorial - Introduction

This tutorial is part of the Essential Kit tutorial series from Voxel Busters.

Welcome to the **Sharing Services** tutorial, your guide to implementing native sharing functionality in Unity cross-platform mobile applications. This feature enables seamless content sharing across iOS and Android platforms using each platform's native interfaces.

## What You'll Learn

In this tutorial series, you'll learn to integrate native sharing capabilities that enhance user engagement and promote viral growth in your Unity mobile games:

- **Mail Composer** - Native email composition with attachments
- **Message Composer** - SMS and messaging functionality  
- **Share Sheet** - Universal sharing to installed applications
- **Social Share Composer** - Direct social media platform integration

## Why Sharing Services Matter for Unity Mobile Game Developers

Sharing functionality is essential for Unity mobile games because it drives user engagement through content sharing, enables viral growth via social networks, facilitates feedback collection, and builds community around your game.

**Key Use Cases:**
- Share high scores and achievements to social media
- Send personalized game invites via email or SMS
- Report bugs with screenshots attached
- Promote user-generated content across platforms

## Prerequisites

**Technical Requirements:**
- Unity 2021.3 or later
- Essential Kit installed and configured
- iOS/Android build support enabled
- Basic C# and Unity development knowledge

## Platform Setup - Essential Kit Advantage

**Essential Kit handles most native setup automatically**, including:
- Framework integration and linking
- Required permission declarations  
- Native library dependencies
- Platform-specific manifest entries

This eliminates complex native configuration work that typically requires separate iOS and Android setup procedures.

**Developers only need to ensure:**
- Test devices have configured email accounts
- Social media apps are installed for testing
- App Store Connect/Google Play Console compliance for distribution
