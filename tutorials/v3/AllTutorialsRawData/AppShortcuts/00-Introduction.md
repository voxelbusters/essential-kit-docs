# App Shortcuts Tutorial - Introduction

This tutorial is part of the Essential Kit tutorial series from Voxel Busters.

## Welcome to App Shortcuts

Welcome to the comprehensive App Shortcuts tutorial for Unity mobile game developers! This tutorial will guide you through implementing dynamic app shortcuts that provide players with quick access to key game features directly from their device's home screen.

## What You'll Learn

In this tutorial series, you'll master:

- Creating dynamic app shortcuts with custom titles and icons
- Managing shortcut lifecycle (adding and removing shortcuts)
- Handling shortcut click events and routing users to specific game content
- Advanced configuration and best practices for Unity mobile games
- Cross-platform implementation for Unity iOS and Unity Android builds

## Why App Shortcuts Matter for Unity Mobile Game Developers

App shortcuts significantly boost player engagement by providing instant access to popular game features. Players can jump directly to daily rewards, continue their progress, access support, or start new challenges without navigating through multiple game screens. This improved user experience leads to higher retention rates and increased session frequency in Unity mobile games.

## Prerequisites

Before starting this tutorial, ensure you have:

- Unity 2021.3.45f1 or compatible version
- Essential Kit properly installed and configured in your Unity project
- Basic understanding of Unity C# scripting
- Target platform set to iOS or Android in Unity Build Settings

## Platform Setup

**Essential Kit handles most native setup automatically!** This includes:
- iOS Quick Actions framework integration
- Android App Shortcuts library configuration
- Required permissions and manifest entries
- Native platform bindings and callbacks

You only need to:
- Configure shortcut icons in Essential Kit Settings (for visual consistency)
- Ensure your app complies with platform-specific app store guidelines
- Test shortcut functionality on target devices

This automated setup saves hours of manual native configuration work!
