# Game Services Tutorial - Introduction

This tutorial is part of the Essential Kit tutorial series from Voxel Busters.

Welcome to the comprehensive Game Services tutorial for Unity mobile game developers! This guide will teach you how to integrate social gaming features like leaderboards, achievements, and player authentication into your Unity iOS and Android games.

## What You'll Learn

- How to authenticate players with Game Center (iOS) and Play Games Services (Android)
- Creating and managing leaderboards for competitive gameplay
- Implementing achievements to boost player engagement
- Managing player profiles and social features
- Integrating server-side authentication for backend connectivity

## Why Game Services Matter for Unity Mobile Game Developers

Game Services transform single-player experiences into social, competitive environments that keep players engaged longer. Features like leaderboards create natural competition between friends, while achievements provide clear goals and progression markers. Player authentication enables cloud save functionality and cross-device gameplay continuity.

Common use cases include:
- Global and social leaderboards for high scores
- Achievement systems for milestone tracking
- Player profile management with display names and avatars
- Friend systems for social gameplay
- Cloud save synchronization across devices

## Prerequisites

- Unity 2021.3 or later
- Essential Kit plugin installed and configured
- Basic understanding of Unity C# scripting
- Test devices for iOS/Android deployment

## Platform Setup

**Essential Kit handles most native setup automatically!** This includes:
- Game Center framework integration (iOS)
- Play Games Services SDK setup (Android) 
- Required permissions and manifest entries
- Native library dependencies

This automation eliminates hours of manual configuration that typically requires:
- Xcode project modifications for iOS Game Center
- Android manifest permission declarations
- Google Play Console configuration synchronization
- Framework linking and build settings

You only need to handle:
- App Store Connect and Google Play Console game setup
- Leaderboard and achievement configuration in respective dashboards
- Privacy policy compliance for your app store listings
